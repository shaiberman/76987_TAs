
---------------------------------------------------------
                   k-Space Tutorial
---------------------------------------------------------

This k-Space tutorial has been developed using MATLAB.

To execute the k-Space Tutorial:

   1) Run MATLAB

   2) Change the current directory to the path
      where you have the files of the tutorial

   3) execute k_space_tutorial from the command line
       (type k_space_tutorial)

   4) although you will find a test image 
       (courtesy of Dr. Marijn E. Brummer, Emory University, Atlanta, GA, USA)
        you can load any bitmap (BMP) image of 256x256 pixels size

         Have Fun !!
